# Healthcare Dataset Analysis: Obesity Prevalence

## Introduction
Obesity is a complex disease involving have too much body fat which was determined as a medical problem causing many other diseases (heart disease, diabetes, high blood pressure, high cholesterolâ€¦). This project utilizes Python and Machine Learning to clean and analyze healthcare data related to obesity. The goal is to identify and predict the key factors that contribute to increasing obesity rates in United States.

## Key Features
* Data cleaning and preprocessing of raw healthcare records.
* Exploratory Data Analysis (EDA) with visualizations (Matplotlib).
* Predictive modeling using Linear Regression and Logistic Regression (Random Forest and Decision Tree).

## Requirements
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Seaborn, Matplotlib
* **Environment:** Google Colab (more preferred), Jupyter Notebook
